Ranch 2.2

0001 01 Jan

Ranch 2.2.0 has been released! Ranch 2.2 is a maintenance release containing a small number of fixes and improvements. The key addition in this release is the support for DTLS which has been added to the test suites. Ranch 2.2 is tested with Erlang/OTP 24.0 onward, although it should be compatible with Erlang/OTP 22.0 and above. It is tested and supported on Linux, macOS and Windows, but should also work on BSD.
